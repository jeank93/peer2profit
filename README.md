# peer2profit
peer2profit via proxy in docker

# Howto (Ubuntu 20.04):

Build:
```
apt update && \
apt upgrade -y && \
apt install docker docker-compose git -y && \
git clone https://github.com/igluko/peer2profit.git && \
cd peer2profit && \
docker build . -t peer2profit \
```

Run container
```
chmod +x run
./run
```

Get containers list:
```
docker ps -a
```
